# Smart Campus REST API

## Overview
This project implements a RESTful API for managing smart campus resources such as rooms, sensors, and sensor readings using JAX-RS.

The API follows REST principles and supports CRUD operations for campus resources.

## Technologies Used

- Java
- JAX-RS (Jersey)
- Maven
- IntelliJ IDEA
- Postman
- GitHub

## Project Structure

src/main/java/com/smartcampus

config – REST configuration  
model – data models  
resource – REST endpoints  
exception – custom exceptions  
filter – request logging filters

## Example Endpoints

GET /api/v1  
GET /api/v1/rooms  
POST /api/v1/rooms  
GET /api/v1/sensors

## How to Run

1. Clone the repository
2. Open the project in IntelliJ
3. Build using Maven
4. Run the application server
5. Test endpoints using Postman
